# CLAUDE.md

语言版本：

- [English](CLAUDE.md)
- [中文](CLAUDE.zh-CN.md)

这个仓库将 Claude Code 作为预期支持的 agent 工作流之一。

Claude 专属指导应尽量保持轻量。默认规则是先遵循仓库共享约定，只有在确有必要时才增加 Claude 特有说明。

## 优先级顺序

1. 先遵循仓库结构和已记录的项目意图。
2. 优先复用共享模板和共享约定。
3. Claude 专属差异必须显式且尽量少。
4. 保持 AI 编写工作流，不要默认按手工编码项目处理。

## 仓库角色

AgentSkeleton 是一个用于以下场景的脚手架项目：

- 新项目初始化
- 历史项目 AI 迁移
- 生成 agent 协作文件
- 检查仓库结构

## 预期结果

Claude Code 应当能在不依赖第二套独立项目体系的前提下，高效地在本仓库中工作。
