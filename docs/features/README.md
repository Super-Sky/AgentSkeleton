# Current Feature Docs

This directory is the current feature-level truth area for AgentSkeleton.

It should describe current capability areas, not one-off release snapshots.

## Current Feature Areas

- CLI workflow orchestration
- host integration
- validation and release hardening

## Current Feature Index

- `feature-cli-workflow.md`
  - current CLI workflow responsibilities and stable command surface
- `feature-host-integration.md`
  - current zero-command host integration responsibilities and artifacts
- `feature-validation-and-release.md`
  - current validation, release hardening, and delivery assets

## Rules

- If a feature is currently supported, summarize it here.
- If a release doc contains current-truth feature detail, that detail should be folded back into `docs/features/`.
- Version directories keep snapshot context, not the main current feature truth.
