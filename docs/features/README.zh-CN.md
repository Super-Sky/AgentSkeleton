# 当前功能文档

本目录是 AgentSkeleton 的当前功能真相区。

这里描述的是当前能力域，而不是某一轮版本快照。

## 当前功能域

- CLI 工作流编排
- 宿主接入
- 验证与发布加固

## 当前功能索引

- `feature-cli-workflow.zh-CN.md`
  - 当前 CLI 工作流职责与稳定命令面
- `feature-host-integration.zh-CN.md`
  - 当前零命令宿主接入职责与产物
- `feature-validation-and-release.zh-CN.md`
  - 当前验证、发布加固与交付资产

## 规则

- 如果某项能力当前已经支持，应优先在这里体现。
- 如果某个版本文档里写的是当前仍然有效的 feature 真相，应把它收口回 `docs/features/`。
- 版本目录保留快照语义，不应承担当前功能主文档角色。
