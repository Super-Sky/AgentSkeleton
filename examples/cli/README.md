# CLI Examples

This directory contains example inputs and outputs for the initial AgentSkeleton CLI contract.

## Included Examples

- `new-project/`: examples for `init-docs`, `plan`, and `next` in a new-project flow
- `legacy-project/`: examples for `reshape-docs`, `plan`, and `next` in a legacy-repository flow

## Purpose

These examples are intended to serve as:

- implementation fixtures
- review artifacts for output shape
- alignment points between the CLI and host models
